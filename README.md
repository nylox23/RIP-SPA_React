# RIP-SPA_React

## Репозиторий для второй части курса «Разработка Интернет Приложений (РИП)» 2025

### Содержит ветки:

#### spa_React - фронтенд перенесен на React, добавлена главная страница и навигационная панель, запросы переписаны на fetch через proxy React
#### spa_deployment - добавлено адаптивное изменение элементов для разных размеров страницы
#### gh-pages - приложение размещено в GitHub Pages
#### tauri - интерфейс приложения переммещен в нативное приложение на tauri
#### spa_auth - фронтенд доработан: добавлена авторизация, страница просмотра и редактирования заявок, методы работы с ними
