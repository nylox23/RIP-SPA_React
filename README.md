# RIP-SPA_React

## Репозиторий для второй части курса «Разработка Интернет Приложений (РИП)» 2025

### Содержит ветки:

#### spa_React - фронтенд перенесен на React, добавлена главная страница и навигационная панель, запросы переписаны на fetch через proxy React
#### spa_InDb - добавлено хранилище и получение данных из PostgreSQL, добавлены методы и элементы фронтенда для работы с БД
#### web_service - первоначальная версия бэкенд веб-сервиса с API для получения кислот и заявок 
#### web_service_auth - доработанная версия бэкенда с добавлением авторизации пользователя по JWT и хранением токенов в Redis
